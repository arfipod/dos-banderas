# Playtesting Guide

## What to Test

During playtests, focus on decisions rather than just win rate.

Record:

- Did the player understand what to do each phase?
- Did the shortcut feel tempting?
- Did the player ever choose Christ's Banner even when the shortcut was attractive?
- Were any cards never played?
- Were any cards always bought?
- Did Fervor become too explosive?
- Did the Final Choice feel fair?
- Did the round tell a recognizable spiritual story?

## Recommended First Session

1. Play 3 solo games on Normal.
2. Download the playtest log after each game.
3. Note the moment where the game felt decided.
4. Check whether the player reached the Final Choice.
5. Check how often manual modifiers were needed.

## Useful Metrics

From logs, inspect:

- Win rate.
- Average Communion after round 5.
- Average Attachment after round 5.
- Number of shortcuts accepted.
- Number of clean victories.
- Number of mixed victories.
- Average Fruits spent.
- Cards bought.
- Cards played most often.
- Cards never played.

## Desired Early Balance

For the first public-feeling prototype:

- Solo win rate: 40–60%.
- Average shortcuts per game: 1–3.
- Most games should reach the Final Choice.
- A clean perfect game should be possible but uncommon.
- At least one painful shortcut decision should occur per game.
